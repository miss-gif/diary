# 감정 일기장

## 프로젝트 소개

    감정 일기장은 일기와 감정을 함께 기록 할 수 있는 서비스입니다.

## 주요 기능:

    일기 작성, 읽기, 수정, 삭제 (CRUD)
    일기 월별 필터
    일기 최신순과 오래된 순 정렬

## 사용 기술 스택:

    React
    react-router
    Vite (빌드 도구)
    Vercel (배포)

## 설치 및 실행

1. 프로젝트를 복제합니다.
   https://github.com/miss-gif/diary
2. 필요한 라이브러리를 설치합니다.
   `npm install`
3. 개발 서버를 실행합니다.
   `npm run dev`
